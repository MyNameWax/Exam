# online_exam_front

本项目基于 Vite + Vue3 + JavaScript，UI 框架为 Ant Design Vue。

## 功能
- 登录页面
- 登录后主页
- 路由管理
- axios 工具类
- 美观布局

## 启动项目

```bash
npm install
npm run dev
```

如遇依赖或 Node 版本问题，请根据提示升级。
